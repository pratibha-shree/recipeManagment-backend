# recipeManagement-backend
```
steps for running project
```
```
git clone https://github.com/pratibha-shree/recipeManagment-backend.git
```
```
cd recipeManagment-backend
```
```
1. Run file at location src/main/resources/database.sql to create recipe database and appropriate tables.
2. Change database.user and database.password in src/main/resource/application.properties to use your  local mysql credentials.
```
```
mvn clean install
```
```
java -jar target\recipie-0.0.1-SNAPSHOT.jar
